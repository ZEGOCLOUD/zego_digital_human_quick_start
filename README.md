# 数字人快速开始项目

这是一个基于 Vue 3 + Vite 的最简配置项目，专为快速开发而设计。

## 技术栈

- **Vue 3** - 渐进式 JavaScript 框架
- **Vite** - 快速构建工具

## 快速开始

### 安装依赖

```bash
npm install
```

### 开发环境

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 项目结构

```
├── public/          # 静态资源
├── src/
│   ├── assets/      # 资源文件
│   ├── components/  # Vue 组件
│   ├── App.vue      # 根组件
│   └── main.js      # 入口文件
├── index.html       # HTML 模板
├── vite.config.js   # Vite 配置
└── package.json     # 项目配置
```
